## 摘要
深度研究智能体（DRAs）将大语言模型（LLM）推理与外部工具相集成。记忆系统使DRAs能够利用历史经验，这对高效推理与自主演化至关重要。现有方法依赖于从记忆中检索相似轨迹以辅助推理，但面临记忆演化效率低下以及存储与检索成本不断攀升等关键局限。为解决上述问题，我们提出了一种新颖的记忆智能体（MIA）框架，该框架采用管理器-规划器-执行器架构。其中，记忆管理器是一种非参数记忆系统，用于存储压缩的历史搜索轨迹；规划器是一种参数化记忆智能体，能够针对问题生成搜索计划；执行器则是另一种智能体，负责在搜索计划的指导下检索并分析信息。为构建MIA框架，我们首先采用交替强化学习范式以增强规划器与执行器间的协作。此外，我们使规划器能够在测试时学习阶段持续演化，其更新过程与推理同步动态进行，且不会中断推理。同时，我们在参数化记忆与非参数化记忆之间建立了双向转换循环，以实现高效的记忆演化。最后，我们引入了反思机制与无监督判断机制，以提升开放世界中的推理与自我演化能力。在十一个基准测试上的广泛实验验证了MIA的优越性。

## Abstract
Deep research agents (DRAs) integrate LLM reasoning with external tools. Memory systems enable DRAs to leverage historical experiences, which are essential for efficient reasoning and autonomous evolution. Existing methods rely on retrieving similar trajectories from memory to aid reasoning, while suffering from key limitations of ineffective memory evolution and increasing storage and retrieval costs. To address these problems, we propose a novel Memory Intelligence Agent (MIA) framework, consisting of a Manager-Planner-Executor architecture. Memory Manager is a non-parametric memory system that can store compressed historical search trajectories. Planner is a parametric memory agent that can produce search plans for questions. Executor is another agent that can search and analyze information guided by the search plan. To build the MIA framework, we first adopt an alternating reinforcement learning paradigm to enhance cooperation between the Planner and the Executor. Furthermore, we enable the Planner to continuously evolve during test-time learning, with updates performed on-the-fly alongside inference without interrupting the reasoning process. Additionally, we establish a bidirectional conversion loop between parametric and non-parametric memories to achieve efficient memory evolution. Finally, we incorporate a reflection and an unsupervised judgment mechanisms to boost reasoning and self-evolution in the open world. Extensive experiments across eleven benchmarks demonstrate the superiority of MIA.

- **核心痛点**：现有记忆系统多采用长上下文检索相似轨迹，存在四大局限：
  1. **演化低效**：记忆仅作为静态上下文注入，缺乏参数化内化机制；
  2. **成本攀升**：历史轨迹不断累积导致存储与检索开销呈指数增长；
  3. **上下文稀释**：长上下文引入噪声，干扰模型对当前问题的注意力；
  4. **记忆类型偏差**：过度关注“事实型记忆”（检索结果），忽视“过程型记忆”（搜索路径、失败尝试、推理策略）。
- **整体含义**：本文提出 **Memory Intelligence Agent (MIA)** 框架，旨在通过解耦记忆管理、规划与执行，构建高效、可演化、低开销的记忆智能体，突破传统长上下文记忆在深度研究任务中的性能瓶颈。

### 2. 方法论：核心思想与关键技术
- **核心架构（Manager-Planner-Executor）**：
  - **Memory Manager（非参数记忆）**：负责存储与检索压缩后的历史轨迹。将图像转为文本描述，将冗长交互抽象为结构化工作流。采用混合检索策略，综合计算语义相似度、历史成功率（价值奖励）与使用频率（频率奖励）。
  - **Planner（参数化记忆/认知中枢）**：基于检索到的正负范例生成逐步搜索计划（CoT），并具备反思与重规划能力。
  - **Executor（执行终端）**：严格遵循Planner的指令，通过ReAct范式调用外部工具进行多轮检索与推理。
- **关键技术细节**：
  - **两阶段交替强化学习（GRPO）**：
    - *阶段1*：冻结Planner，训练Executor理解并执行计划。奖励函数由答案正确性（70%）、工具调用规范性（20%）与格式合规性（10%）加权组成。
    - *阶段2*：冻结已训练的Executor，训练Planner吸收记忆上下文、优化规划与反思策略。奖励函数额外引入反思触发合理性（5%）与中间答案正确性（20%）。
  - **测试时在线学习（TTL）**：推理过程中同步进行探索、记忆提取与参数更新。Planner生成多组计划轨迹，通过组相对优势计算（$\hat{A}_i = (R_i - \mu_R)/(\sigma_R + \epsilon)$）在线更新参数，实现非参数记忆（上下文）与参数记忆（权重）的双向转换。
  - **无监督同行评审判断机制**：在无标准答案的开放场景中，模拟学术审稿流程。设置三位专项审稿人（逻辑一致性、信源可信度、结果有效性）与一位领域主席（AC），通过多维度正交评估与元决策，生成近似监督信号以驱动自演化。

### 6. 主要结论与发现
- **性能跃升**：MIA 显著提升前沿闭源模型表现（如 GPT-5.4 在 LiveVQA/HotpotQA 分别提升 9% 和 6%）。
- **小模型越级**：基于 Qwen2.5-VL-7B 的轻量 Executor 结合 MIA，平均准确率提升 31%，大幅超越未增强的 Qwen2.5-VL-32B（领先 18%）。
- **记忆机制优越性**：全面超越现有 SOTA 记忆框架（平均 +5%），证明“过程导向规划”优于“长上下文事实堆砌”。
- **协同优化有效**：交替 RL 使 Planner 与 Executor 策略对齐，响应长度随数据集特征自适应收敛。
- **无监督自演化可行**：在无标签设置下，MIA 性能接近监督版，且随 TTL 迭代轮次（Epoch 1→3）持续稳定上升，验证了开放世界自主进化能力。

### 7. 优点（方法/实验亮点）
- **架构解耦创新**：将非参数记忆检索与参数化规划分离，从根本上缓解长上下文噪声与存储瓶颈。
- **在线持续学习**：TTL 机制实现“推理-更新”无缝并行，无需中断工作流即可吸收新经验。
- **评估机制突破**：首创类学术同行评审的无监督判断框架，通过多维度正交评估与元决策，有效缓解传统 LLM-as-a-judge 的“幻觉客观性”问题。
- **强泛化与可扩展性**：同时赋能 7B 开源模型与顶级闭源模型，且工具链仅需基础图文搜索即可取得 SOTA 表现，证明性能增益源于记忆架构而非复杂工具堆砌。

### 8. 不足与局限
- **任务覆盖局限**：当前主要聚焦深度检索与问答任务，尚未验证在代码执行、具身控制或高动态交互环境中的适用性。
- **算力与延迟开销**：无监督评审与 TTL 在线更新需频繁调用 32B 参数模型作为 Manager/Judger，推理延迟与算力成本较高，可能限制实时性要求严苛的部署场景。
- **执行器冻结策略**：TTL 仅更新 Planner 参数，Executor 保持冻结。在分布外极端场景下，可能限制端到端工具调用策略的自适应能力。
- **工具依赖风险**：框架性能高度依赖外部搜索工具的可用性与返回质量，未深入探讨工具失效、网络延迟或对抗性检索结果下的鲁棒性机制。
